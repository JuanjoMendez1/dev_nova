Landing Page o Página de Aterrizaje de la empresa Maplins. Destaca brevemente la información clave de la empresa, presenta una sección de galería visualmente atractiva, permitiendo a los visitantes explorar visualmente los productos con los que se cuenta. Ademas, se ha integrado un formulario de contacto intuitivo que facilita a los usuarios solicitar más información de manera rápida y sencilla. Proporcionamos diversas formas de comunicación para garantizar una conexión fluida con la empresa.

Bajo la denominación 'Entre Nos', se presenta un sistema destinado a la programación de citas, la visualización de información relacionada con las mismas y el seguimiento de pagos asociados. Este sistema integra un protocolo de autenticación diseñado para limitar el acceso exclusivamente a usuarios debidamente registrados en la base de datos. Una vez autenticados, los usuarios tendrán acceso a distintos módulos, entre los que se encuentran: Usuarios, Doctores, Pacientes, Citas y Dashboard. Cada uno de estos módulos cumple con una función específica, permitiendo el registro, la edición y la eliminación de información en la base de datos correspondiente.
